    **src/lib/api/validation.ts:**
    Create validation helpers:
    ```typescript
    export function validateNarrative(text: string): { valid: boolean; error?: string } {
      if (!text || text.trim().length < 20) return { valid: false, error: 'La narrativa debe tener al menos 20 caracteres' }
      if (text.length > 10000) return { valid: false, error: 'La narrativa no puede superar los 10000 caracteres' }
      return { valid: true }
    }

    export function validateEvidenceType(type: string): boolean {
      return ['narrative', 'pasted_chat', 'screenshot', 'url', 'username', 'alias', 'phone', 'note'].includes(type)
    }
    ```

    **src/app/api/cases/route.ts:**
    POST handler that:
    1. Parse JSON body
    2. Validate narrative with validateNarrative()
    3. Generate public_id (8 chars from UUID)
    4. Call createCase(publicId) from repositories
    5. Track analytics event 'case_started'
    6. Return { case_id, public_id, status: 'received' }

    If validation fails, return 400 with { error: string }.
    If creation fails, return 500 with { error: 'Internal error' }.
    Log all errors with logger.ts.

    ```typescript
    export class ApiError extends Error {
      constructor(public statusCode: number, message: string) {
        super(message)
      }
    }

    export function handleApiError(error: unknown): Response {
      if (error instanceof ApiError) {
        return Response.json({ error: error.message }, { status: error.statusCode })
      }
      log('error', 'Unhandled API error', { error })
      return Response.json({ error: 'Internal server error' }, { status: 500 })
    }
    ```

    **src/middleware.ts:**
    Basic rate limiting by IP:
    - Track requests per IP in memory (Map<string, { count: number, resetAt: number }>)
    - Allow 100 requests per minute per IP
    - Return 429 with { error: 'Too many requests' } when exceeded
    - Reset counter after 60 seconds

    Apply middleware to API routes.
