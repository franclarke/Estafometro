    ```typescript
    export async function matchOfficialPatterns(
      caseId: string,
      signals: CaseSignal[],
      caseType?: string
    ): Promise<PatternMatch[]> {
      // 1. Load all patterns
      // 2. For each pattern:
      //    a. Calculate coverage: % of core_signals detected
      //    b. Calculate high_weight_match: % of high_weight_signals detected
      //    c. If caseType matches pattern.category, boost score
      //    d. match_score = (coverage * 0.6) + (high_weight_match * 0.4)
      // 3. Keep matches with score >= 30
      // 4. Store pattern_matches via storePatternMatches()
      // 5. Return matches
    }
    ```

    Pattern matching is additive — it strengthens existing signals but doesn't override hard rules.
